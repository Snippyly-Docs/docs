## When to use

The comment bubble is a button that can open the comment associated with some target element. It also shows the total number of threads in the given comment.

## How to use

### Add the comment bubble component

Import the component:

```tsx copy
import { SnippylyCommentBubble } from '@snippyly/react';
```

Then add it somewhere in your app:

```tsx copy showLineNumbers {6}
import { SnippylyCommentBubble } from '@snippyly/react';

export const SomeComponent = () => {
    return (
        ...
        <SnippylyCommentBubble targetCommentElementId="TARGET_ELEMENT_ID" />
        ...
    )
}
```

## Customization

### Give it a custom trigger button

**Example**
```html copy
<SnippylyCommentBubble>
   <button>Bubble button</button>
</SnippylyCommentBubble>
```
