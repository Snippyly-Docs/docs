# Popover mode

_Show popovers instead of comment pins._

---

{/* <Comment style="margin-top: 2.5rem" user={"Rachel"} documentId={commentDocumentId} height={500}/> */}

## When to use

If you want to display popover style comments instead of our default freestyle comment pins.

## How to use

### Enable popover mode

You can use our client API to enable popover mode:

```js copy
const commentElement = client.getCommentElement();
commentElement.enablePopoverMode(true);
```

**Example**

```tsx copy {8-21} showLineNumbers
import React, {useEffect} from "react";
import { useSnippylyClient } from '@snippyly/react';

const SomeComponent = () => {
  
  const { client } = useSnippylyClient();


  useEffect(() => {
    if (client) {
      enablePopoverMode();
    }
  }, [client]);

  const enablePopoverMode = () => {
    if (client) {
      const commentElement = client.getCommentElement();
      commentElement.enablePopoverMode(true);
    }
  }

  
  return (
    //... rest of the component code...
  )
}

export default SomeComponent;
```
